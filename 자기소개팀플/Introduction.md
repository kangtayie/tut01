# 📄 프로젝트 결과 개요서: 인터랙티브 팀 소개 웹페이지

> **포켓몬스터 스탯 시스템을 활용한 인터랙티브 팀원 소개 웹페이지 구축 프로젝트**

---

## 1. 프로젝트 개요
* **프로젝트명**: 포켓몬스터 스탯 시스템 활용 팀 소개 웹페이지
* **목적**: 팀원 3인의 성향, 업무 방식, 장단점을 게임 스킬 및 스탯 형태로 수치화하여 직관적으로 전달
* **개발 환경 및 기술 스택**:
    * **IDE**: Visual Studio Code (Live Server)
    * **Languages**: `HTML5`, `CSS3`, `JavaScript (Vanilla)`
    * **Assets**: 투명 배경(PNG) 합성 이미지, 커스텀 폰트, 웹 UI 소스

## 3. 화면 UI 및 기능 구현 명세
"오늘의 포켓몬은 뭘까요?" 블라인드 테스트 형식을 차용하여 구현되었습니다.

### 3.1 메인 레이아웃
* **배경**: 연구소 콘셉트의 배경 이미지 렌더링
* **타이틀**: 상단 중앙 텍스트 섀도우 효과 적용
* **모니터 영역**: 중앙 팀원 이미지 디스플레이 영역 할당

### 3.2 주요 기능
1.  **실루엣 효과**: 
    - CSS `filter: brightness(0)`를 활용하여 초기 로드 시 검은 실루엣 출력
2.  **정체 확인 토글(Toggle)**:
    - JavaScript 이벤트 리스너를 통해 필터 해제 및 정보 패널(스탯) 호출
3.  **배열 순회 네비게이션**:
    - `Array`에 저장된 팀원 객체 데이터를 인덱스 기반으로 순회하며 DOM 업데이트
    - 페이지 전환 시 실루엣 상태 초기화 기능 포함
